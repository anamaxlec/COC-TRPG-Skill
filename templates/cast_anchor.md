## 写入记忆建议

```bash
python scripts/memory.py add --name "<团名>" --section protagonist --text "<主角人设锚点>"
python scripts/memory.py add --name "<团名>" --section cast --text "<NPC公开锚点>"
python scripts/memory.py add --name "<团名>" --section secret --text "<NPC隐藏动机或秘密>"
```
