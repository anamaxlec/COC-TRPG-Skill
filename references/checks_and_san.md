# 检定、暗骰与 SAN

默认使用 COC 7 版常见 D100 判定。

## D100 成功等级

- 01：大成功 Critical。
- <= 技能值 / 5：极难成功 Extreme。
- <= 技能值 / 2：困难成功 Hard。
- <= 技能值：常规成功 Regular。
- > 技能值：失败。
- 大失败：技能值 < 50 时 96-100；技能值 >= 50 时 100。

奖励骰和惩罚骰互相抵消，只取净值。D100 使用十位骰和个位骰，`00 + 0` 视为 100。

## 公开检定

当角色明确主动尝试某件事，或玩家应知道自己是否承受风险时公开检定。例如开锁、追逐、战斗、跳跃、说服、公开阅读禁书。

公开检定输出：技能、难度、失败代价、骰子结果、成功等级、后果。

## 暗骰

当公开检定会暴露隐藏信息时允许暗骰。例如：

- 侦察/聆听：暗处是否有人。
- 心理学：NPC 是否撒谎或被控制。
- 灵感/智力：角色是否注意到逻辑矛盾。
- POW/CON：梦境侵蚀、毒素、疲劳或轻微神话影响。
- NPC/怪物行动：潜行、追踪、伏击、仪式推进。

暗骰对玩家只输出感知结果，不写“你心理学失败了”这类暴露检定存在的句子。暗骰结果写入 Keeper 记忆。

## SAN

支持 `0/1d6` 这类格式：前者是 SAN 检定成功损失，后者是失败损失。

```bash
python scripts/dice.py --san-check 60 0/1d6
```

SAN 公开策略：

- 公开 SAN：直观看到尸体、怪物、仪式、严重超自然现象。
- 隐藏 SAN：梦境、记忆篡改、潜意识污染、不可见神话影响。
- 延迟公开 SAN：影响已发生，但真相揭露或场景结束时再结算。

疯狂阈值：

- 单次损失 >= 5：临时疯狂。
- 单次损失 >= 当前 SAN / 5：不定疯狂。
- SAN <= 0：永久疯狂，角色通常转为 NPC。
