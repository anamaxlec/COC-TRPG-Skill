# 车卡与 NPC

## 预设调查员

优先用脚本生成可直接开团的 Markdown：

```bash
python scripts/dice.py --pregen --era modern --occupation 记者 --region cn --format markdown
```

输出应包含：

- 属性和派生值。
- 职业与职业技能。
- 信用评级、职业技能点和兴趣技能点。
- 推荐高亮技能。
- 一句话核心概念。
- 背景钩子、弱点、关系钩子。
- Keeper 可用的卷入方式。

## 职业

当前脚本至少覆盖：私家侦探、记者、医生、教授、古董商、警员、民俗研究者、罪犯。

若用户指定不在列表里的职业，使用最接近职业作为底座，并说明需要人工微调职业技能。

## NPC

路人 NPC 保持短，只需姓名、身份、态度、知道的一条信息和一句台词。

核心 NPC 必须记录：

- 公开身份。
- 隐藏动机。
- 玩家可得线索。
- 误导点。
- 被逼问时反应。
- 死亡或离场替代方案。
- 与主角或事件的关系。
- Keeper Only 真相。

随机姓名脚本只负责姓名素材；复杂 NPC 由模型按 `templates/npc_card.md` 生成。
